Coherence Optimization Operator

With polarimetric data alone it is hard to discriminate different natural media such as forest with trees of different types. Recent studies show that discrimination of different natural media could be achieved with interferometric data. The result of the classification based on optimal Pol-InSAR coherence set shows an enhanced description and understanding of the scattering from trees of different types in the observed forest area. This operator computes a set of three optimal coherence from a pair of Pol-InSAR products.

Input and Output

  • The input to this operator is a stack of coregistered master and slave bands.
  • The output of this operator is a set of three optimal coherence bands.

Parameters Used

   The following processing parameters are used for this operator:
  • Window Size: size of sliding window for computing mean coherency matrix.
  • Output optimum coherence in complex checkbox: if selected, complex data are output for the three optimum polarimetric interferometric coherences.
