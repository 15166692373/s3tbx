Processor Description - Land/Water Mask

User Interface

The Land/Water Mask processor can be invoked in the Sentinel Toolbox from tool menu by selecting Optical->Preprocessing->Land/Water Mask. On the command line the Land/Water Mask processor is available by means of the Graph Processing Tool gpt which is located in the Sentinel Toolbox bin directory. Typing gpt LandWaterMask -h displays further information.
