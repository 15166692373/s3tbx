Frequently Asked Questions

Installation F.A.Q.

What is the minimum required hardware to run the Sentinel Toolbox

It is recommended to have at least 4GB of memory. 

To run the 3D WorldWind View, it is recommended to have a 3D graphics card with updated drivers.

The software has been tested to work on 32 and 64 bit Windows XP, Vista, 7, 8 and Linux.

I can't start up the Toolbox

Try running from the command line to see any errors using dat.bat on Windows or dat.sh on Linux. If the error mentions something to do with any of "WorldWind, native library, JOGL, glugen, or OpenGL" then it may be a problem with WorldWind:

Try first to update your drivers (How to update windows drivers) (ATI/AMD - Nvidia - Intel)

If that doesn't work, then try removing WorldWind by deleting the file from your installation folder /modules/s1tbx-world-wind-view.jar

When I start the DAT it says it could not create the Java virtual machine

In dat.bat or dat.sh try reducing the Java maximum heap size -Xmx4G value to somthing smaller.

How can I get WorldWind to work? I see a blank window in the World View.

To use the WorldWind World Map and 3D View, a 3D video card with updated drivers is necessary.
Update your video card drivers. (How to update windows drivers) (ATI/AMD - Nvidia - Intel)

How can I move the .s1tbx folder somewhere else?

By default the toolbox creates a windows_user_home_folder\.s1tbx (~/.s1tbx on Linux)

The .s1tbx folder is used to store temporary data such as preferences, logs, etc.
 
If you would like to change the location of this folder:

Edit the file InstallFolder\config\s1tbx.config

change the line
#s1tbx.application_tmp_folder = c:\\AuxData

to
s1tbx.application_tmp_folder = your_new_path

How to change the AuxData folder

Edit the file InstallFolder\config\s1tbx.auxdata.config

Change the AuxDataPath value to your new path and restart the Toolbox.
